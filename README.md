# Sitemap-For-Typecho

## 功能

1、为`Typecho`生成`sitemap`包含**首页、独立页面、分类、标签、文章**

2、推送百度搜索资源平台，目前仅支持普通收录推送，卑微的我没快速收录权限

不支持发布文章自动更新`sitemap`及自动推送，可能会影响文章发布速度所以没有做支持

## 更新
